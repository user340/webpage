インプットメソッド——ibus-mozc

ibus-mozc

ibus-mozcはその名前のとおり「ibus」と「mozc」2つのソフトウェアから成ります。

ibusはIntelligent Input Busを意味する名前で、Linux OS向けのインプットフレームワーク（input framework）です。一方mozcはマルチプラットフォームの日本語のインプットメソッドエディタ（Input Method Editor, IME）です。インプットメソッド（input method）は文字を入力するためのソフトウェアのことで、日本語や中国語・韓国語といったキーボードで直接入力できない文字を書くために使われます。ibus-mozcは、ibusフレームワークで開発されたmozcということになります。

インストール

pkgsrc(7)ではinputmethod/ibus-mozcパッケージとして提供されています。ibus-mozcパッケージに依存しているgypパッケージがPython 3.4/3.5/3.6/3.7に対応していないため、マクロPYTHON_VERSION_DEFAULTは27を指定する必要があります。

# cd /usr/pkgsrc/inputmethod/ibus-mozc
# make PYTHON_VERSION_DEFAULT=27 install clean clean-depends

$HOME/.xinitrc

X11起動時にibus-mozcを動かすには$HOME/.xinitrc内のウィンドウマネージャを動かす行の前に以下を加えてください。

export XMODIFIERS="@im=ibus"
export GTK_IM_MODULE="ibus"
export QT_IM_MODULE="ibus"
export DefaultIMModule="ibus"
ibus-daemon -drx
